# Smart Task Scheduler 📋

A feature-complete desktop task manager built in Java that automatically 
prioritizes your work using a Priority Queue data structure.

## Features
- ✅ Add, edit, delete, and complete tasks
- 🔺 Four priority levels: Low, Medium, High, Critical
- ⏰ Automatic 15-minute deadline reminders
- 🔍 Live search + 6 smart filters (Today, This Week, High Priority, Overdue, etc.)
- 💾 Auto-save/load via built-in JSON engine (no external libraries)
- 📊 Live summary sidebar with task stats

## Tech Stack
- Java SE 11+ | Java Swing | PriorityQueue | java.util.Timer

## Run
javac SmartTaskScheduler.java
java SmartTaskScheduler
